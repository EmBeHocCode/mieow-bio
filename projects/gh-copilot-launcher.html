Giới thiệu

EmBeby-Mieow-DEV là công cụ launcher giúp bạn sử dụng GitHub Copilot CLI một cách trực quan thông qua giao diện đồ họa thay vì chỉ dùng command line.

Tool này được build bằng C# WPF, hỗ trợ Windows 10/11, giúp người mới bắt đầu dễ tiếp cận AI coding assistant mà không cần nhớ command phức tạp.

Lưu ý: Tool hiện đang trong giai đoạn phát triển và chưa hoàn thiện. Một số tính năng có thể chưa ổn định hoặc đang được cải thiện. Vui lòng feedback nếu gặp lỗi!

Yêu cầu hệ thống

Bạn cần chuẩn bị những gì trước khi dùng

Windows 10/11

Hệ điều hành Windows 10 hoặc Windows 11 với kết nối Internet ổn định.

Tài khoản GitHub

Tài khoản GitHub có quyền sử dụng GitHub Copilot (subscription hoặc free trial).

GitHub CLI

Cài đặt GitHub CLI từ cli.github.com và đăng nhập tài khoản.

Hướng dẫn cài đặt nhanh

Bước 1: Cài GitHub CLI

1. Mở link chính thức: https://cli.github.com/
2. Tải bản cài đặt Windows và cài xong.
3. Mở PowerShell mới, chạy: 

gh --version

Bước 2: Đăng nhập GitHub

Chạy lệnh sau và làm theo hướng dẫn: 

gh auth login

Chọn: GitHub.com → HTTPS → Login with a web browser

Bước 3: Cài extension Copilot

Kiểm tra xem có extension chưa: 

gh copilot --help

Nếu báo lỗi "không tìm thấy lệnh copilot", cài thêm: 

gh extension install github/gh-copilot

Bước 4: Chạy launcher

1. Mở file EmBeby-Mieow-DEV.exe
2. Màn hình Setup Data Folder sẽ hiện ra
3. Chọn nơi lưu data (settings/logs/chat transcripts)

Lưu ý: Đường dẫn phải có ChatCLI. Nếu bạn nhập D:\, tool sẽ tự thêm thành D:\ChatCLI.

Bước 5: Sử dụng

1. Ở Project folder bấm Browse và chọn thư mục code
2. Dùng các nút:
- Open Copilot CLI
- Run /create feature
- Run /fix bug
- Run /refactor code
- Change Data... (đổi lại nơi lưu data)

Tính năng chính

Những gì tool này làm được

Giao diện đồ họa

Không cần gõ command, chỉ cần click chuột để mở Copilot CLI và thực hiện các tác vụ phổ biến.

Quick Prompts

Các nút nhanh cho /create, /fix, /refactor giúp tiết kiệm thời gian gõ lệnh và nhớ cú pháp.

Quản lý Project

Browse và chọn project folder dễ dàng, tool tự động lưu lại path để lần sau không cần chọn lại.

Chat Logs

Tự động lưu chat transcript vào <DataRootPath>\chat-logs\ với tên file theo thời gian.

Data Management

Cho phép thay đổi data folder bất cứ lúc nào qua nút "Change Data..." trong giao diện chính.

Path Validation

Kiểm tra và tự động thêm "ChatCLI" vào path nếu thiếu, đảm bảo cấu trúc folder đúng chuẩn.

Lỗi thường gặp và cách xử lý

"gh was not found"

- Cài lại GitHub CLI từ https://cli.github.com/
- Mở lại terminal mới rồi kiểm tra gh --version

"prompt file not found"

- Bạn đang chọn sai Project folder
- Repo cần có .github/prompts/... để dùng 3 nút quick prompt

Không tạo được data folder

- Bấm "Change Data..."
- Thử path đơn giản: D:\ChatCLI\EmBeby-Mieow-DEV\data

Tool mở nhưng không chat được

- Kiểm tra gh auth status
- Kiểm tra quyền Copilot trên account GitHub của bạn

Thiếu .NET runtime

Launcher build theo kiểu self-contained false, máy người dùng cần runtime phù hợp.
Tải tại: https://dotnet.microsoft.com/en-us/download/dotnet

Setup nhanh trong 5 bước

1) Cài GitHub CLI: https://cli.github.com/

2) Đăng nhập: gh auth login

3) Chạy file EmBeby-Mieow-DEV.exe

4) Chọn data folder (tool tự thêm ChatCLI nếu thiếu)

5) Chọn repo và bấm Open Copilot CLI
