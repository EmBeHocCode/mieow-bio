Portfolio đang chờ Vite khởi chạy

Bạn đang mở app bằng kiểu static server như Live Server 127.0.0.1:5500, nên file src/main.tsx không được bundle và trang sẽ trắng.

Chạy local đúng cách: npm install rồi npm run dev

Nếu muốn up host tĩnh, hãy dùng nội dung trong thư mục dist/ sau khi chạy npm run build.
